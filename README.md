## COMMANDS

### Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

And see it running at: [http://localhost:3000/](http://localhost:3000/)



### Compile and Minify for Production

```sh
npm run build
```
